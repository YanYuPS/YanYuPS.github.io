发表于 2018-01-10 | 分类于 计算机基础

ASC||码，UTF-8编码

ASC||码

  1. 计算机中所有信息都可以表示为一个二进制字符串，每个二进制位有0 1两种状态
  2. 1字节对应8位二进制数，也就是1字节可以表示256种状态
  3. ASC||码：
    • 描述英语中的字符和8位二进制的对应关系
    • 一共定义了128个字符，用到了后7位，第一位统一为0
    • 常用：【0-9】对应48-57，【A-Z】对应65-90，【a-z】对应97-122

unicode

  1. 世界上的所有语言的所有字符，都用一个数字来表示
  2. 字符集（比ASC||码更全面，ASC||码仅记录了英文128个字符，unicode记录了所有字符）

unicode编码方案

  • UTF-8可变长（常用）
  • UTF-16基础平面用两个字符，辅助平面用四个字符
  • UTF-32四个字符

UTF-8编码

  1. 编码规则：
    Unicode 十六进制码点范围 UTF-8 二进制
    0000 0000 - 0000 007F 0xxxxxxx
    0000 0080 - 0000 07FF 110xxxxx 10xxxxxx
    0000 0800 - 0000 FFFF 1110xxxx 10xxxxxx 10xxxxxx
    0001 0000 - 0010 FFFF 11110xxx 10xxxxxx 10xxxxxx 10xxxxxx
  2. 特点：可变长
  3. 1个字节：第一位为0，后面七位对应这个字符的unicode码点，同ASC||码
  4. n个字节（n>1）：
    • 第一个字节的前n设置位1，第n+1位设置为0
    • 后面字节的前两位设置为10
    • 剩下的二进制位，用这个字符的unicode码点来填充
  5. 例子
    “汉”的 Unicode 码点是 0x6c49（110 1100 0100 1001），通过上面的对照表可以发现，0x0000 6c49 位于第三行的范围，那么得出其格式为 1110xxxx 10xxxxxx 10xxxxxx。接着，从“汉”的二进制数最后一位开始，从后向前依次填充对应格式中的 x，多出的 x 用 0 补上。这样，就得到了“汉”的 UTF-8 编码为 11100110 10110001 10001001，转换成十六进制就是 0xE6 0xB7 0x89。
